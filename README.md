## How to Use

1. **Upload Your Materials**: Click the upload area and select your notes, slides, or study materials
2. **Choose Learning Method**: Select the learning style that works best for you
3. **Generate Learning Plan**: Click to create a personalized study plan
4. **Start Teaching Session**: Begin your interactive learning experience
5. **Navigate Lessons**: Use Previous/Next buttons to move through content
6. **Complete & Restart**: Finish your session or restart to try a different method

## Getting Started

Simply open `index.html` in your web browser to start using The Classroom.

For local development with a server:
```bash
python3 -m http.server 8080
# Then navigate to http://localhost:8080/index.html
```
